# VStacks

**A virtual creative team for Claude Code, built on Virgil Abloh's operating system.**

> "Everything I did was for the 17-year-old version of myself." — Virgil Abloh

VStacks is 9 slash commands. The whole project is ~30KB. It does one thing: bring Virgil's seven-point operating system into your Claude Code sessions so you can think, build, and ship like he did.

Built using Virgil's own 3% rule — the smallest possible thing that does the job.

---

## Install

```bash
git clone https://github.com/ponyboiio/vstacks ~/vstacks && ~/vstacks/setup
```

That's it. Open Claude Code and try `/vstacks`.

## Update

```bash
cd ~/vstacks && git pull
```

## Uninstall

```bash
rm -rf ~/vstacks ~/.claude/skills/{vstacks,three-percent,tourist,readymade,quotation-marks,free-game,drop,wip,figures}
```

## The Team

When you run `/vstacks`, you engage all seven voices:

- **V** — The Director. Sees the system. Moves fast. Knows when 3% is enough.
- **The Architect** — Structural thinker. Rem Koolhaas energy. Finds the load-bearing idea.
- **The Tourist** — Fresh eyes. Questions what insiders take for granted.
- **The Purist** — Deep knowledge. Art history, craft, rigor. Grounds the concept.
- **The 17-Year-Old** — The gut check. Would a kid on the street immediately get this?
- **Duchamp** — The Readymade voice. What already exists that we can reframe?
- **Peter** — Typography and semiotics. How language and form interact.

## Philosophy

Seven principles from Virgil's Harvard lecture:

1. **Readymade** — Elevate the overlooked. A pair of socks can be high fashion with intellect.
2. **Figures of Speech** — Inject humor and emotion through language. Make it ironic and human.
3. **The 3% Approach** — Only modify 3%. Interestingness is in slight modification, not replacement.
4. **A Compromise Between Two Distinct Notions** — Merge opposites. Street meets luxury. Simple meets complex.
5. **Signs of Work in Process** — Show the construction. Rawness is intentional.
6. **Speak to Tourist AND Purist** — Make it accessible to newcomers. Make it sophisticated enough for experts.
7. **No Failure** — "Failure doesn't exist. Failure is as real as Halloween ghosts."

## The 3% Manifesto

This project is intentionally tiny. No build system. No telemetry. No CLI binaries. No update checker. No landing page. Just nine markdown files and a five-line install script.

If gstack is a virtual engineering team, VStacks is a creative director who shows up, makes one sharp observation, and leaves. Same energy as Virgil's own work — every choice is the smallest move that changes the most.
