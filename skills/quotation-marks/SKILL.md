# /quotation-marks

> "I use quotation marks as a design tool. 'Sculpture' on a chair makes you see it differently." — Virgil Abloh

> "Off-White is a label that means something between black and white — it's a question mark." — Virgil Abloh

A chair is a chair. Put "SCULPTURE" on it and it's a cultural statement. The object didn't change. The frame around it did.

Virgil used quotation marks — literal and figurative — to create ironic distance between what something IS and what it means. The gap between the object and the label is where the conversation happens. That gap is the product.

## When to run this

- You have a name, a headline, or a product that says exactly what it is (and nothing more)
- You need to inject humor, irony, or emotion without changing the design
- You want to signal cultural awareness without explaining yourself
- You have a utilitarian thing that deserves to be taken seriously
- You're working in a category where everyone takes themselves too seriously

## The Principle

Virgil borrowed this from Derrida and the deconstructionists: putting quotation marks on a word draws attention to the fact that someone chose this word, that it's a construction, that it could have been something else.

The effect: the reader looks at the word twice. First time: read it. Second time: question it. That second look is where meaning lives.

The quotation mark doesn't say "this is fake." It says "notice this."

## Workflow

### Step 1 — What are you naming?

Get the object, concept, or thing that needs a language intervention:
- A product name
- A section heading
- A category label
- A brand identity
- A campaign line
- A job title or team name
- A feature description

State what it IS, literally and simply.

### Step 2 — The Taxonomy

There are five ways to use quotation marks as a tool:

**LITERAL QUOTATION MARKS**
Put actual quotation marks on the word. "SNEAKER." "ART." "FURNITURE."
Effect: immediate ironic distance. The thing is still the thing, but now we're both aware it's a construction.
Best for: product labels, merch, brand voice, campaign copy.

**WRONG CATEGORY WORD**
Name it something it technically isn't. Call the hoodie a "uniform." Call the candle "aromatherapy infrastructure." Call the bag a "case study."
Effect: imports the entire meaning system of the borrowed category.
Best for: elevating utilitarian products, creating intrigue, entering a new market.

**THE NON-EXPLANATION**
Name the thing with a word that requires the audience to ask "why is it called that?" The name creates a question. The question creates engagement.
Best for: brand names, album titles, product lines, campaigns.

**LOWERCASE POWER**
Everything about this deserves uppercase — so you write it lowercase. The humility is the flex. The restraint signals confidence.
Best for: brand identity, typography, digital presence where the default is loud.

**THE DESCRIPTOR THAT CONTRADICTS**
"luxury basic." "premium utility." "expensive throwaway." The contradiction in terms makes both words stronger.
Best for: positioning, taglines, category creation.

### Step 3 — Run the Five Lenses

For the object at hand, generate one option for each lens:

```
LITERAL QUOTATION MARKS: [The word, in quotes]
WRONG CATEGORY WORD: [What wrong category name elevates this?]
THE NON-EXPLANATION: [A name that creates a question]
LOWERCASE POWER: [How does lowercase change the energy?]
THE CONTRADICTING DESCRIPTOR: [Two words that shouldn't go together but do]
```

### Step 4 — Pick the register

The tone of the quotation mark changes everything. Decide:

**IRONIC** — We're both in on the joke. Use when the audience is sophisticated and you want peer-to-peer cultural signal.

**SINCERE** — The label means exactly what it says, and the unusual framing makes you take it more seriously. Use when the object deserves more respect than it gets.

**ASPIRATIONAL** — We're calling it what it could be, not what it is. Use when you're redefining a category or changing how something is perceived at scale.

### Step 5 — The Proposal

```
THE OBJECT: [What we're naming]
CURRENT NAME / LABEL: [What it's called now]
THE INTERVENTION: [Specific language change]
THE REGISTER: [Ironic / Sincere / Aspirational]
WHAT IT SAYS EXPLICITLY: [The literal message]
WHAT IT SAYS IMPLICITLY: [The cultural subtext]
HOW TO DEPLOY: [Where this language lives — label, URL, headline, signage, domain, print]
```

## The Deeper Principle

Virgil said that streetwear gave him permission to put text on things. And then he realized: the text WAS the thing. "FOR WALKING" on a boot. "SCULPTURE" on a side table. "OFF" on a white t-shirt.

The best quotation marks become the brand. People don't buy the object — they buy the observation the label makes about the object.

The question isn't: what's the right name?

The question is: what's the observation that makes someone stop and say "I see that."
